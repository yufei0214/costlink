# CostLink 技术文档

## 1. 项目概述

CostLink 是一个费用报销系统，覆盖「用户提交报销」到「管理员审核与打款」的完整流程。

核心能力：
- LDAP/本地模拟登录
- 凭证图片上传
- OCR 金额识别（支持人工修正）
- 报销状态流转与后台审批
- 已付款记录图片批量导出

## 3. 系统架构

### 3.1 逻辑分层

1. **展示层（Frontend）**
   - 负责页面渲染、表单交互、路由守卫、Token 管理。
   - 统一通过 `src/api/index.ts` 调用后端 `/api`。

2. **业务层（Backend Service）**
   - 负责认证授权、报销业务规则、OCR 调用、状态流转、导出逻辑。

3. **数据访问层（Mapper + Entity）**
   - 使用 MyBatis-Plus 映射业务实体到 MySQL 表。

4. **基础设施层（MySQL / 文件存储 / LDAP / OCR Provider）**
   - MySQL 存储结构化数据。
   - 上传凭证图片落盘到 `UPLOAD_PATH`，通过 `/api/uploads/**` 访问。
   - LDAP 提供企业身份认证。
   - OCR 服务提供金额识别能力。

### 3.2 运行拓扑

- 本地开发：
  - 前端 Vite 默认 `:5173`，代理 `/api` 到 `http://localhost:8080`
  - 后端 Spring Boot 默认 `:8080`
- Docker 部署：
  - `frontend` 容器（Nginx）
  - `backend` 容器（Spring Boot）
  - `mysql` 容器（MySQL 8.0）

## 4. 关键模块说明

## 4.1 前端模块

- **API 层**：`costlink-frontend/src/api/index.ts`
  - 请求拦截器注入 JWT
  - 响应拦截器处理 401（清理登录态并跳转登录页）
- **路由守卫**：`costlink-frontend/src/router/index.ts`
  - `requiresAuth`、`requiresAdmin` 控制访问边界
- **状态管理**：`costlink-frontend/src/stores/user.ts`
  - 管理用户信息与认证状态

## 4.2 后端模块（`com.costlink`）

- **controller**：对外 REST API（Auth / User / Reimbursement / Admin）
- **service**：业务编排（认证、OCR、审批、账户维护、导出）
- **mapper + entity**：数据库映射
- **config**：JWT、Security 过滤链、跨域、分页配置
- **exception**：统一异常处理

---

## 5. 安全与权限模型

- 系统为 **无状态 JWT** 鉴权。
- 公共接口：
  - `/api/auth/login`
  - `/api/auth/logout`
  - `/api/uploads/**`
- 权限控制：
  - `/api/admin/**` 仅管理员可访问
  - 其他 `/api/**` 默认需要登录
- 管理员身份来源：`ADMIN_USERS` 环境变量。
- LDAP 支持两种模式：
  - 用户 DN 模式（simple bind）
  - 服务账号查询 + 用户绑定模式
- `LDAP_ENABLED=false` 时可走本地/模拟认证流程。

## 6. 核心业务流程

### 6.1 用户报销流程

1. 登录系统
2. 维护收款账号
3. 上传凭证图片
4. OCR 识别金额（可手动修正）
5. 提交申请（状态进入 `PENDING`）

### 6.2 管理员处理流程

1. 查看仪表盘与待处理列表
2. 审核申请：
   - 通过：`PENDING -> CONFIRMED`
   - 驳回：`PENDING -> REJECTED`
3. 打款后标记：`CONFIRMED -> PAID`
4. 可对已付款记录执行图片批量导出 ZIP

## 7. 数据模型（核心表）

- `t_user`：用户与基础资料
- `t_reimbursement`：报销主单
- `t_reimbursement_image`：报销图片明细
- `t_admin_config`：管理端相关配置

---

## 8. 配置说明

后端配置文件：`costlink-backend/src/main/resources/application.yml`（支持环境变量覆盖）

关键变量：
- 数据库：`SPRING_DATASOURCE_URL` / `SPRING_DATASOURCE_USERNAME` / `SPRING_DATASOURCE_PASSWORD`
- 认证：`JWT_SECRET`
- LDAP：`LDAP_ENABLED` / `LDAP_URL` / `LDAP_BASE` / `LDAP_USER_DN_PATTERN` / `LDAP_BIND_DN` / `LDAP_BIND_PASSWORD`
- 管理员：`ADMIN_USERS`
- OCR：`QWEN_API_KEY` / `QWEN_MODEL` / `QWEN_ENDPOINT` / `QWEN_TIMEOUT`
- 上传：`UPLOAD_PATH`

上传限制：
- 单文件 10MB
- 单请求 50MB
